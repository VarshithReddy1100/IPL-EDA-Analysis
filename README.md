# IPL-EDA-Analysis
A complete data science case study on IPL data featuring end-to-end EDA, data cleaning, statistical insights, visualizations, and an interactive Tableau dashboard. This project explores trends in wins, venues, toss outcomes, and top performers using Python and Tableau.

🏏IPL Data Analysis (EDA + Tableau Dashboard)

A complete data science exploration of IPL dataset using Python, Pandas, Matplotlib, Seaborn & Tableau.

📌 Project Overview

This project performs a detailed Exploratory Data Analysis (EDA) on IPL data to uncover trends, performance insights, and hidden patterns related to teams, players, toss decisions, match outcomes, and venues.

It includes:
✔ Jupyter Notebook (Python EDA)
✔ Tableau Dashboard for Visual Insights
✔ Final Report (30+ pages)

🔍 Key Insights Identified

Teams winning the toss do not always win the match.

Batting-second teams have a higher win percentage in many seasons.

Mumbai Indians are the most consistent winning team across seasons.

Certain venues favor chasing, while others favor batting first.

Top performers contributing to match victories were identified.
